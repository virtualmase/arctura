P1 · SOVEREIGNTY

Authority does not flow from the chain — it flows through it. Each node holds a sealed mandate the network can verify but never override.

P2 · COHERENCE

Byzantine partitions resolve to a single coherent reality. No reorganizations, no rollbacks — only forward attestation.

P3 · ATTESTATION

Every action is signed

Provenance is a first-class output, not an afterthought. Each action emits a Merkle-anchored proof readable by any verifier on or off-chain.

P4 · RESONANCE

Consensus is not just safe and live — it is in-tune. Resonance adapts block cadence to network coherence, not the other way around.

Compute follows the cleanest grid. Nodes earn weight not only by stake but by the sustainability of the energy they consume.

Frequency FAQ

Direct answers, citation-ready.

What is Arctura?

Arctura is a high-frequency decentralized subnet for autonomous resource management. It coordinates verifiable agent action across Earth-bound nodes using a six-layer signal stack and a Byzantine-fault-tolerant consensus called Resonance.

What does “Arcturian-class intelligence” mean?

It refers to a design lineage prioritizing wisdom-aligned coordination, attested provenance, and benevolent autonomy. The metaphysical Arcturian-blue frequency — clarity, healing, transformation — expressed as protocol invariants.

What are the five primitives?

P1 Sovereignty (every node owns its mandate), P2 Coherence (state convergence under load), P3 Attestation (every action is signed and verifiable), P4 Resonance (BFT tuned to network frequency), and P5 Stewardship (carbon-aware scheduling).

How is Arctura different from a normal blockchain?

Legacy consensus orders transactions. Arctura attests mandates. The unit of work is a sovereign agent action with verifiable provenance, not a key-pair-signed transfer. There are no re-orgs — only forward attestation.

How do I connect a node?

Activation is a three-phase protocol: Tune (signal audit), Bond (mandate chain registration with attested keys), Ascend (cluster join with live attribution wiring). Provisioning completes within one consensus cycle.

Is Arctura open-source?

The kernel is Apache-2.0. Reference clients, MCP bindings, and the Truth Ledger schema are all public. Mandate marketplace components ship in Phase 04.
